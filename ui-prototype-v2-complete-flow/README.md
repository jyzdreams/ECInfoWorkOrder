# EC 工單系統 v2 完整流程原型

本資料夾為獨立的靜態 React 原型，包含流程確認包、全角色頁面效果圖與可切換使用者的互動演示。

## 交付內容

- `docs/flow-confirmation.md`：全角色泳道、6 狀態、履約與消息提醒流程。
- `docs/page-index.md`：13 張效果圖與流程節點對照。
- `docs/page-effects/`：PC 與師傅端的靜態評審圖。
- `src/App.jsx`：可重置的本地演示資料、角色權限、消息、時間線與全流程互動。

## 演示路徑

切換角色依序完成：CS 建單 → Team Leader 分派 → 師傅接單／打卡／初檢 SR → 報價確認 → 採購到貨 → Team Leader 分派安裝任務 → 梁師傅安裝 SR／完工單 → 採購確認 → 會計開票與收款。

可於 CS 工單詳情取消尚未完成的工單；師傅可拒單退回待分派；報價組可模擬無需採購或未中標。左下「重置演示資料」可回到初始場景。
