# 全角色頁面效果圖與流程節點索引

效果圖由原型的角色工作台直接截圖，檔案置於 `docs/page-effects/`。每張圖均保留角色、當前用戶、狀態和待辦資料。

| 序號 | 效果圖 | 角色 | 對應流程 |
|---:|---|---|---|
| 01 | CS 首頁 | CS | 建單、催單、服務追蹤 |
| 02 | CS 新建工單 | CS | 建單 → 待分派 |
| 03 | Team Leader 分派中心 | TL | 待分派、拒單重派、安裝重派 |
| 04 | 師傅移動端初檢任務 | 師傅 | 接單、打卡、初檢 SR |
| 05 | 師傅初檢 SR | 師傅 | 提交初檢 SR、是否需報價 |
| 06 | 報價工作台 | 報價組 | 建立、修訂、客戶確認／未中標 |
| 07 | 採購與物流 | 採購部 | PO、物流、到貨、完工單確認 |
| 08 | 師傅移動端安裝任務 | 師傅 | 安裝接單、安裝 SR、完工單 |
| 09 | 開票與收款 | 會計部 | 單次開票、單次收款、歸檔 |
| 10 | 消息提醒中心 | 全角色 | 未讀數、指定接收人、跳轉與已讀 |
| 11 | 全鏈路看板 | 主管 / 管理員 | 審計、權限與統計 |
| 12 | 老闆經營總覽 | 老闆 | 履約風險與收款 |
| 13 | 流程確認包 | 全角色 | 泳道、狀態、履約與消息 |

所有頁面使用繁體中文、深色側欄、白色資料卡和藍色流程主色；師傅端使用 390px 寬度移動框。`qa-*` 檔案為驗證證據，不計入評審頁面索引。
