# 页面清单

## 1. PC管理后台

| 页面 | 使用角色 | 核心功能 | 一期 |
|---|---|---|---|
| 工单列表 | CS、CR、Leader、报价、采购、财务、管理层 | 搜索、筛选、状态查看、导出、批量操作入口。 | 是 |
| 新建工单 | CS、CR | 来电/WhatsApp录入、附件上传、PM/CM选择、客户/地址选择。 | 是 |
| 工单详情 | 全角色按权限 | 基本信息、状态流转、照片、处理记录、报价/采购/报销关联。 | 是 |
| 派单工作台 | Team Leader | 待分派、改派、工单池确认、拒单处理、团队负载。 | 是 |
| 客户/地址管理 | 管理员、CS主管 | 客户、地址、区域、不可拍照配置、英文别名。 | 是 |
| 团队/区域配置 | 管理员、管理层 | 区域码、团队、Leader、成员、技能标签。 | 是 |
| 报销关联列表 | 财务、师傅、Leader | 选择已完成工单、上传票据、查看报销状态。 | 简版 |
| 采购关联列表 | 采购、Leader | PO、ML、预计/实际到货、材料状态。 | 简版 |
| 报价关联列表 | 报价、Leader | 报价编号、负责人、状态、附件。 | 简版 |
| 操作日志 | 管理员、审计/管理层 | 查看创建、派单、接单、状态、导出日志。 | 是 |

## 2. 师傅移动端

首屏原则：不超过3个核心入口，不展示后台菜单。来源：M2。

| 页面 | 核心控件 | 说明 |
|---|---|---|
| 我的待办 | 今日/待接/处理中三段 | 只显示本人或本组可操作工单。 |
| 工单卡片 | 地点、联系人、问题、是否可拍照、按钮 | 按钮：接单、拒绝、到场、上传、完成。 |
| 接单/拒绝 | 接单按钮、拒绝原因 | 拒绝需填写原因，如正在执行其他PM。 |
| 到场签到 | 到场按钮、地图选点、时间记录 | 不做实时轨迹。 |
| 拍照上传 | 现场前、现场后、材料/设备、其他 | 不可拍照客户自动提示无需照片。 |
| 处理结果 | 结果、需报价、需材料、需客户确认、备注 | 支持最小文字输入，语音转写后置。 |
| 历史记录 | 近期待办/已完成 | 只看本人范围。 |

## 3. Team Leader工作台

| 页面 | 功能 |
|---|---|
| 待派工单 | 查看CS提交未派工单，按区域/类型/优先级排序。 |
| 团队负载 | 查看本团队师傅接单、处理中、今日完成。 |
| 工单池审核 | 审核主动接单申请。 |
| 拒单处理 | 查看拒绝原因并重新分派。 |
| 待确认完成 | 核查照片、结果、报价/材料标记，确认或退回。 |
| Outstanding原因 | 按等待材料、等待报价、客户确认、施工中分类。 |

## 4. 管理层看板

| 看板 | 指标 |
|---|---|
| 总览 | 今日新增、待处理、处理中、已完成、超时。 |
| 区域/团队 | 1P/2P/4P/5P/6P/7P待办、完成率、超时数。 |
| Outstanding | 未完成原因分类、最长等待天数、负责人。 |
| 工单质量 | 照片完整率、不可拍照数量、退回补充数。 |
| 报价/采购关联 | 等待报价、等待材料、PO到货状态。 |
| 报销关联 | 已关联工单金额、待审批、退回。 |

## 5. 报销页面

| 页面 | 一期范围 | 二期增强 |
|---|---|---|
| 新建报销 | 选择月份、申请人、关联工单、上传票据、填写金额。 | OCR识别、明细拆分、自动排除公众假期。 |
| 报销列表 | 按申请人、月份、状态筛选。 | 个人排行、团队费用趋势。 |
| 报销详情 | 查看票据、关联工单、审批状态。 | 部分驳回、补充资料流。 |

## 6. 采购/报价预留页面

| 页面 | 一期范围 | 二期增强 |
|---|---|---|
| 采购关联 | PO编号、ML编号、材料说明、预计/实际到货、附件。 | Material Request、供应商、收货、扫码、库存联动。 |
| 报价关联 | 报价编号、报价负责人、状态、附件、金额。 | 报价明细、版本控制、客户确认、PDF导出。 |

## 7. 页面优先级

| 优先级 | 页面 |
|---|---|
| P0 | 新建工单、工单列表、工单详情、师傅我的待办、师傅工单卡片、派单工作台、基础权限、操作日志。 |
| P1 | 工单池、Outstanding细分看板、超时提醒、简版报销关联、管理层增强看板。 |
| P2 | 报价编号/状态、PO编号/ML编号、采购到货状态、客户自助查询编号字段预留。 |
| P3 | OCR、客户外部查询、公开链接、AI搜索。 |
