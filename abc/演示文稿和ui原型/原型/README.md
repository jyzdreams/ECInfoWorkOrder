# ECInfo × DingTalk 項目啟動會 HTML 簡報

本目錄為客戶第一次匯報使用的 20 頁 HTML 簡報原型，使用純 HTML / CSS / JavaScript 實現，無後端、無框架依賴。

## 文件結構

- `index.html`：20 頁可編輯簡報內容
- `styles.css`：16:9 畫布、PPT 風格版式、藍綠協同視覺
- `app.js`：上一頁 / 下一頁、鍵盤切換、目錄跳轉、全螢幕展示
- `assets/logos/`：ECInfo 與 DingTalk Logo 本地素材
- `assets/reference/`：當前 20 頁 PNG 效果圖，僅作視覺參考

## 本地預覽

直接用瀏覽器打開 `prototype/index.html` 即可預覽。

操作方式：

- 點擊底部 `‹` / `›` 切換頁面
- 使用鍵盤左右鍵、PageUp / PageDown 切換頁面
- 在第 2 頁點擊目錄項跳轉章節
- 點擊 `全螢幕` 或按 `F` 進入全螢幕展示
