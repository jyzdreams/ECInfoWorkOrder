# ECInfo 工單原型繁體化與業務邏輯測試記錄

日期：2026-07-02

## 測試範圍

- 原型文件：`ui-prototype/index.html`
- 語言檢查：客戶可見文案、狀態、按鈕、通知、模擬資料、移動端文案
- 業務流程：CS 建單/受理/取消、Team Leader 分派/確認、師傅接單/拒單/提交、報價頁狀態、移動端首屏

## 已完成調整

- 將原型客戶可見文案統一為繁體中文與香港客戶常用口徑。
- 將本地緩存 key 切換為 `ecinfo_hk_*_v2`，避免瀏覽器舊簡體模擬資料影響測試。
- 補充 CS 對「待分派」工單的「取消」操作，符合未派工單可取消的業務口徑。
